Docker Machine

Thu, Apr 12, 2018

Docker Machine is a tool that lets you install Docker Engine on virtual hosts, and manage the hosts with docker-machine commands.

Docker-machine Installation

Docker-machine Commands

  • docker-machine active: See which machine is “active”
  • docker-machine config: Print the connection config for machine
  • docker-machine create: Create a machine.
  • docker-machine env: Set environment variables
  • docker-machine inspect: Inspect information about a machine
  • docker-machine ip: Get the IP address of one or more machines.
  • docker-machine kill: Kill (abruptly force stop) a machine
  • docker-machine ls: List machines
  • docker-machine mount: Mount directories from a machine to your local host, using sshfs.
  • docker-machine regenerate-certs: Regenerate TLS Certificates for a machine
  • docker-machine restart: Restart a machine
  • docker-machine rm: Remove a machine
  • docker-machine scp: Copy files from your local host to a machine, from machine to machine, or from a machine to your local host using scp
  • docker-machine ssh: Log into or run a command on a machine using SSH.
  • docker-machine start: Start a machine
  • docker-machine status: Get the status of a machine
  • docker-machine stop: Gracefully Stop a machine
  • docker-machine url: Get the URL of a host
