### Git configuration

```node
git config --global user.name "Your Name"  
git config --global user.email "you@example.com"
git config --global color.ui auto 
```


### STAGE

```node
git status
git add [file]
git reset [file]
git diff
git diff --staged
git commit -m “[descriptive message]”
```
